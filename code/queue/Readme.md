## 队列

### 1.基本概念

队列（queue）也是一种线性表。
- 它的特性是先进先出（FIFO）；
- 只允许在队尾插入元素、队头删除元素。
- 就像排队一样，刚来的人入队（push）要排在队尾（rear），每次出队（pop）的都是队首（front）的人。

### 2.常见队列类型
- 单链队列
- 循环队列

### 3.常用两种存储结构
- 顺序存储结构
- 链式存储结构


### 4.常见基本操作
- init: 初始化队列
- empty: 判断队列是否为空
- clear: 清空队列元素
- length: 获取队列长度
- **push: 入队**
- **push: 入队**
